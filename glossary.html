Glossary

App

\ Noun.

App is a short term for “application”. It is a program or a software that has a specific purpose. It can be run on a web browser, on a computer, or on a mobile phone. The term “app” is mostly used for mobile programs.

Antivirus software

\ Noun.

Antivirus software is a program that prevents, detects, and removes malware on your device.

Bandwidth

\ Noun.

Bandwidth is the maximum amount of data that can be transferred on a specific network or Internet connection in a specific amount of time. For instance, a network can have a bandwidth of 300 megabytes per seconds. This means that this network cannot transmit more than 300MB in one second.

Hard drive

\ Noun.

A hard disk drive (HDD) stores data on your device. The data is stored magnetically on a stack of disks that the hard drive can read.

Incognito mode

\ Noun.

Incognito mode is a mode in the browser Google Chrome. While using this method, you’re browsing and download history isn’t recorded. Also, it prevents cookies from being stored. Furthermore, it doesn’t use the current cookies of the normal mode. This means that the incognito mode isn’t connected to any accounts, and it can be perceived as a different device by many websites.

IP address

\ Noun.

An IP (Internet Protocol) address identifies a device on a network. The identifier is a unique string of numbers like “174.94.148.79”.

Memory

\ Noun.

Memory is the temporary data that is stored close to the motherboard in a computer, mobile phone, or any other device. It is data can be accessed quickly by the system. Unlike a hard drive, all the information in the memory is erased when the device is turned off.

Operating system

\ Noun.

An operating system is a software that controls the computer's basic function such as the processing of programs, and the assignment of storage in memory. Windows, Ubuntu (Linux), Fedora (Linux), and Mac OS are operating systems.

SMS

\ Noun.

SMS is an abbreviation for “Short Message Service”. It is a term for text messages.
