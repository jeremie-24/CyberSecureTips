Glossary

App

\ Noun.

"App" is a short term for “application”. It is a program or software that has a specific purpose. It can be run on a web browser, on a computer, or on a mobile phone. The term “app” is mostly used for mobile programs.

Antivirus software

\ Noun.

Antivirus software is a program that prevents, detects, and removes malware on your device.

Bandwidth

\ Noun.

Bandwidth is the maximum amount of data that can be transferred on a specific network or Internet connection in a specific amount of time. For instance, a network can have a bandwidth of 300 megabytes per second. This means that this network cannot transmit more than 300MB in one second.

Hard drive

\ Noun.

A hard disk drive (HDD) stores data on your device. The data is stored magnetically on a stack of disks that the hard drive can read.

Incognito mode

\ Noun.

Incognito mode is a mode in the browser Google Chrome. While using this method, you’re browsing and download history isn’t recorded. Also, it prevents cookies from being stored. Furthermore, it doesn’t use the current cookies of the normal mode. This means that the incognito mode isn’t connected to any accounts, and it can be perceived as a different device by many websites.

IP address

\ Noun.

An IP (Internet Protocol) address identifies a device on a network. The identifier is a unique string of numbers like “174.94.148.79”.

Memory

\ Noun.

Memory is the temporary data that is stored close to the motherboard in a computer, mobile phone, or any other device. It is data that can be accessed quickly by the system. Unlike a hard drive, all the information in the memory is erased when the device is turned off.

Operating system

\ Noun.

An operating system is software that controls the computer's basic functions such as the processing of programs and the assignment of storage in memory. Windows, Ubuntu (Linux), Fedora (Linux), and Mac OS are operating systems.

SMS

\ Noun.

SMS is an abbreviation for “Short Message Service”. It is a term for text messages.

Program

\ Noun.

A program is a collection of instructions that makes a device to something. For instance, a program could play a video on the device’s screen.

Network

\ Noun.

Software

\ Noun.

Software is the term used for computer programs.

System

\ Noun.

Systems are sets of programs designed to accomplish a certain task, for example websites, softwares, computers or video games.

Data leak

\ Noun.

Data leaks are when sensitive information is unintentionally made available on the Internet.

Hashing

\ Noun.

Hashing is the process of taking a key, usually a password, and then processing it into an encrypted result. Usually, this process is made to be impossibly reversible, meaning you can get the same result from the same key, but not the key from the result.

Database

\ Noun.

Databases are systems used by most websites and softwares to store data like emails, names, addresses, etc.

Cyberattack

\ Noun.

Cyberattacks are attacks that target systems. The goal of these attacks vary, although they usually aim to disrupt or breach these systems.

Brute Forcing

\ Noun.

Brute forcing is a kind of cyberattack that targets encrypted passwords. The attack tries to find the correct password of a given encryption by guessing multiple passwords until it finds the correct password that encrypts to the same encryption.

Ransomware

\ Noun.

Ransomware is a type of malware that locks the device of a user until a ransom is paid. These ransoms are usually monetary.

Encryption

\ Noun.

Encryption is a process used to protect sensitive data by using functions to obfuscate the original data so it can't readily be used.

Vulnerability Exploitation

\ Noun.

Vulnerability exploitation is a type of cyberattack that uses the existing flaws of a system to attack it.

Encryption Cracking

\ Noun.

Encryption cracking is the process of finding the original message that was encrypted in an encryption.
